# 🕵️ Credit Control Anomaly Detector – Learn R GUI & 3D Visualization

[![R Version](https://img.shields.io/badge/R-%E2%89%A54.0-blue.svg)](https://www.r-project.org/)

> **Learn how to build a complete credit control tool in R** – from data cleaning to anomaly detection and an **interactive 3D GUI** that lets you explore financial statements like never before.

This repository teaches you how to:
- ✅ Load & clean messy Excel statements  
- ✅ Automatically detect 5 types of payment anomalies  
- ✅ Create **interactive 3D plots** (rotate, zoom, hover)  
- ✅ Export shareable HTML dashboards and CSV reports  
- ✅ Understand R GUI concepts (`tcltk`, `shiny`, `plotly`)
