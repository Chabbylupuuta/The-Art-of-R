# 🎨 Learn R GUI – Point‑and‑Click Applications in R

[![R](https://img.shields.io/badge/R-%E2%89%A54.0-blue.svg)](https://www.r-project.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> 📌 **This repository will teach you how to build and use Graphical User Interfaces (GUI) in R.**  
> No more command‑line only – create real desktop apps or interactive dashboards with just a few lines of code.

---

## 🧠 What you’ll learn

- ✅ **Built‑in GUI toolkits** – `tcltk`, `gWidgets2`
- ✅ **Web apps with Shiny** – dashboards anyone can use in a browser
- ✅ **Standalone point‑and‑click tools** for data analysis
- ✅ **How to add buttons, file pickers, plots, and tables** to your R scripts

---

## 📂 Repository contents

| File / Folder | Description |
|---------------|-------------|
| `gui_basics.R` | Simple `tcltk` example – a window with a button |
| `shiny_app/` | Interactive dashboard for credit control (from the anomaly detector) |
| `gwidgets_demo.R` | Demonstrates `gWidgets2` with input fields and graphs |
| `data/` | Sample Excel statement to test with |
